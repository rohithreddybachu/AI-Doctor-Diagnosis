# AI Doctor Diagnosis System

An AI-powered Doctor Diagnosis System built using HTML, Node.js, Prolog, and JSON.

## Project Overview

This project helps users:

* Enter symptoms
* Get possible disease predictions
* View medicine suggestions
* See approximate medicine costs
* Place medicine orders
* Save patient history
* Use voice input for symptoms

---

## Features

* Symptom-based disease prediction
* AI diagnosis system
* Medicine recommendation
* Billing system
* Patient record storage
* Order placement system
* Voice symptom input
* Payment options
* JSON database storage

---

## Technologies Used

* HTML
* CSS
* JavaScript
* Node.js
* Prolog
* JSON

---

## Project Files

```text
index.html        → Website frontend
server.js         → Node.js backend server
server.pl         → Prolog diagnosis logic
patients.json     → Patient data storage
orders.json       → Order history storage
start-website.cmd → Start command file
```

---

## How to Run Project

### Step 1

Install Node.js from:

```text
https://nodejs.org
```

### Step 2

Open project folder.

### Step 3

Open terminal inside folder.

### Step 4

Run:

```bash
node server.js
```

### Step 5

Open browser and visit:

```text
http://localhost:8080
```

---

## Project Workflow

1. User enters symptoms
2. AI checks symptom database
3. Disease prediction is generated
4. Suggested medicines appear
5. Medicine cost is calculated
6. Order can be placed
7. Data is saved in JSON files
